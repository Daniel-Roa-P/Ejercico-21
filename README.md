# Ejercicio Simulación 21.

El actual repositorio trae con sigo un ejercicio en el que se realiza una simulación del popular juego de azar denominado como BlackJack en donde el jugador competirá contra la mesa para ver cuál de los dos se acerca más al número 21, esto lo realiza con la particularidad de hacer uso de la programación funcional, lo que trae con sigo la imposibilidad de usar estados y de esta forma evitar los efectos de lado.
